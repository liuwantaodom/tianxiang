# 陈四

  

## 身份

  

城南蚕市蚕种摊贩，位置偏僻（蚕市尽头，挨着茅厕），摊位冷清。

  

## 背景

  

四十来岁的瘦男人，下巴上一撮山羊胡子。面前摆着七八盒蚕种，旁边还搁着几捆桑条。

  

识货且有商业头脑，是[[苏锦婳]]最早的蚕种供货渠道之一。

  

## 关键情节

  

| 章节 | 事件 |

|------|------|

| 第3章 | [[苏锦婳]]用七文钱+野蚕丝团抵价+告知陈种盐水催青补救法，换得其三龄蚕种。陈四后告知"下回蚕市逢五开市，你要是还来，找我" |

| 第17章 | [[苏锦婳]]至其铺中购湖种四盒（每盒30文，共120文+杭州脚钱6文）。陈四告"湖种今年俏，有人订了六盒走了" |

| 第19章 | 陈四告其用盐水催青法救回陈种，出蚁率回六成；赠嘉兴嫁接桑种子（出叶厚蚕爱吃） |

  

## 技术能力

  

陈四本身具备一定蚕桑知识，但对催青技术不如[[苏锦婳]]精通。

  

[[苏锦婳]]传授的**盐水催青法**让他能将陈种出蚁率从报废边缘拉回六成，说明他对新技术愿意接受并执行到位。

  

## 与苏锦婳的关系

  

早期供货商关系，[[苏锦婳]]以技术换资源的模式在他身上得到验证：

  

```

[[苏锦婳]]提供技术知识（盐水催青）→ 陈四提供蚕种渠道 → 形成稳定供货关系

```

  

陈四是[[苏锦婳]]在蚕种供应链上的第一个固定合作方，为后续杂交育种实验埋下伏笔。
