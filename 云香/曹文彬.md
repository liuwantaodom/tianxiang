# 曹文彬

  

## 身份

  

江宁织造，内务府汉姓包衣出身。

  

## 性格特征

  

- 精明圆滑，长袖善舞

- 善于投机，看准机会果断出手

- 识货且有底线，并非唯利是图

  

## 核心作用

  

第一个看懂[[苏锦婳]]丝绸**政治价值**的官方人物。

  

凭借敏锐的政治嗅觉和商业头脑，成为连接[[苏锦婳]]商业帝国与宫廷的关键桥梁。

  

## 关键节点

  

| 节点 | 说明 |

|------|------|

| **画舫宴** | 与主角二人组（[[沈砚之]]、[[苏锦婳]]）的第一次正式接触 |

| **[[天香纱]]问世** | 担着风险将[[天香纱]]进献至御前，奠定其在宫廷与商业之间的独特地位 |

  

## 商业路径

  

```

江南（丝产地）→ 江宁织造（曹文彬）→ 内务府（宫廷）→ 京城

```

  

通过与[[沈砚之]]的合作，打通了**江南→京城→内务府**的关键通道。

  

## 人物定位

  

曹文彬是[[沈砚之]]和[[苏锦婳]]**早期最重要的政治投资人与商业合作伙伴**。

  

他不仅是商业上的伙伴，更是政治上的知音，为[[苏锦婳]]丝绸进入宫廷提供了关键的政治背书与渠道支持。
