# 嘉庆帝（颙琰）

  

## 身份

  

- 乾隆第十五子（颙琰）

- 清朝第五位皇帝

- 年号嘉庆（1760-1820）

  

## 政治立场

  

- 整顿内务府，清算[[和珅]]

- 重用实干汉臣

- 支持丝绸产业与技术革新

- 是[[沈砚之]]和[[苏锦婳]]最终的政治靠山

  

## 与主角的关系

  

| 人物 | 关系描述 |

|------|----------|

| vs [[沈砚之]] | 真正赏识其实干能力与忠诚；但[[沈砚之]]本质上是其"满汉制衡"棋局中的一枚重要棋子 |

| vs [[苏锦婳]] | 信任其技术与判断；[[苏锦婳]]的不可替代性（[[天香纱]]核心技术的唯一掌握者）反而比[[沈砚之]]更强 |

  

## 小说定位

  

嘉庆帝不是被主角"操控"的愚君，而是有自己的政治判断和手腕的**成熟帝王**。

  

与主角是**互相需要、互相利用**的关系，而非简单的君臣主从。

  

## 核心逻辑

  

```

主角需要 → 嘉庆的政治背书

嘉庆需要 → 主角的实干能力 + [[天香纱]]的不可替代性

```

  

两人之间没有忠诚，只有利益交换——这是清代皇权与臣子关系的真实写照。
