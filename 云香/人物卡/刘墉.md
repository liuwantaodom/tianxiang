# 刘墉

  

## 身份

  

- 军机大臣

- 大学士（正一品）

- [[沈砚之]]的官场恩师

- 汉臣领袖之一

  

## 性格特点

  

- 清正刚直

- 老谋深算

- 深谙满汉权斗之道

- 对乾隆朝政局有清醒而悲观的认知

  

## 政治作用

  

| 作用 | 说明 |

|------|------|

| 提拔 | 提拔[[沈砚之]]入仕途 |

| 教导 | 教[[沈砚之]]朝堂规则与满汉平衡术 |

| 改革 | 联手推行丝税改革和清丈旗田 |

| 警钟 | 失势后成为[[沈砚之]]的"政治警钟"——让他提前看到汉臣在皇权棋盘上的最终命运，加速政治成熟 |

  

## 经典语录

  

> "在这朝堂上，汉臣从来都是棋子，区别只在于是谁的棋子。"

  

## 人物定位

  

刘墉是[[沈砚之]]仕途路上最重要的引路人。

  

但他与[[沈砚之]]的关系本质上是**师生而非父子**——他会教导，但不会为学生牺牲自己的政治前途。当利益冲突时，刘墉会像棋子一样被牺牲，而[[沈砚之]]必须从他的结局里学会如何在棋盘上活得更久。
