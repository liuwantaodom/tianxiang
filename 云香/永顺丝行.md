# 永顺丝行

  

## 身份

  

清河镇丝行，收购生丝。

  

## 关键人物

  

**吴掌柜**：永顺丝行掌柜，负责验货议价。见[[沈砚之]]秀才腰牌后态度不冷不热。

  

## 关键情节

  

| 章节 | 事件 |

|------|------|

| 第15章 | [[苏锦婳]]与[[沈砚之]]到永顺丝行售丝。吴掌柜验货后报价三两二钱（实际成交价）。[[苏锦婳]]对外报一两八钱以封锁消息 |

  

## 商业位置

  

永顺丝行是[[苏锦婳]]早期最重要的变现通道：

  

```

蚕茧 → 缫丝 → 永顺丝行（变现）→ 资金回笼 → 扩大生产

```

  

首笔成交即实得 **三两二钱**，是全书主角完成原始资本积累的关键一步。
