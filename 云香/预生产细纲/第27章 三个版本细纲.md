## 版本A：砚之远行

**基调**：沈砚之离村前的最后一日——他把所有能交代的事情全部当面交接，苏锦婳第一次独自面对丝局运营的开局之夜。

### 核心情节点

**情节点1** [定制收契书·落笔]

**沈砚之线**
- **男主状态**：锦云丝局前厅，桌案铺开契书格式
- **核心事件**：沈砚之将"议定交收"条款正式写入契书模板，并当着苏锦婳的面逐条解释措辞含义——"议定"给蚕农留面子，"交收"让丝行挑不出毛病
- **冲突**：苏锦婳发现契书里有一处漏洞——没有写明"茧子品级由谁判定"
- **反转**：沈砚之补了一句："由你定。但写的是'双方议定'。"——他懂她的意思：品级判定权本质上是定价权
- **钩子**：沈砚之放下笔："这契书，你收好。我走之后，你拿着它去找蚕户谈。"

**情节点2** [账目交接·分文不差]

**苏锦婳线**
- **女主状态**：锦云丝局账房，两人对账
- **核心事件**：沈砚之把开业以来的每一笔收入、支出、欠款、预付款全部列清交给苏锦婳。数字精确到文。
- **冲突**：账面上有一笔"欠永顺丝行三两"的应付款——吴掌柜的预付款还没还完
- **反转**：沈砚之临走前说："吴掌柜那边，我已经打好了招呼。你拿着丝去，他不会压价。"
- **钩子**：沈砚之走后第一天，苏锦婳要独自去谈第一批定制收

**情节点3** [织娘磨合·三十梭的差距]

**苏锦婳线**
- **女主状态**：后院织房，李桂香婆媳上机
- **核心事件**：苏锦婳考核年轻媳妇的手艺，发现她织了三十梭后手法开始走样——踏板节奏不稳，经线张力不均。她当场纠正，要求重来
- **冲突**：李桂香护短，说"头回上手，慢些正常"
- **反转**：苏锦婳没有退让，让年轻媳妇把手伸出来——指尖有泥，没洗干净就上机了
- **钩子**：苏锦婳定下规矩："机前洗手，这是第一条铁律。违反一次，扣十文。"

**情节点4** [街头偶遇·石青初现]

**苏锦婳线**
- **女主状态**：锦云丝局门外，傍晚
- **核心事件**：苏锦婳送走最后一批蚕户后，在门口看见一个乞丐——衣衫褴褛，但眼神锐利，跟其他乞丐不一样。他没有伸手讨饭，只是站在墙角，看着丝局的招牌
- **冲突**：陈四路过，低声说了一句"这人在这儿站了三天了"
- **反转**：苏锦婳让陈四去问，对方不答，只接过饭走了
- **钩子**：苏锦婳记住了这个人的眼神——像是在等什么

**情节点5** [砚之启程·临别一刻]

**沈砚之线 + 苏锦婳线**
- **核心事件**：沈砚之启程赴江宁。行囊简单：一件换洗青衫、一罐墨、几十张纸。二两银子盘缠贴身藏好
- **核心台词**：沈砚之在门口回头，对苏锦婳说："丝局的事，撑不住就等我回来。"苏锦婳答："你只管考你的。"
- **钩子**：沈砚之的背影消失在街口。沈诚之站在苏锦婳身边，问："砚之什么时候回来？"苏锦婳说："等他中了举人。"

### 核心台词（3-5句）

1. 沈砚之："由你定。但写的是'双方议定'。"（品级判定权）
2. 沈砚之："吴掌柜那边，我已经打好了招呼。"
3. 苏锦婳："机前洗手，这是第一条铁律。违反一次，扣十文。"
4. 陈四（低声）："这人在这儿站了三天了。"
5. 苏锦婳："你只管考你的。"

### 情绪曲线

[平稳→紧绷→意外→平静→淡淡的别离]

定制收契书签完、账目交接清楚、织娘规矩立下——三条线都在推进。最后以沈砚之启程收尾，别离情绪淡淡的，但暗含重量。

### 主题落点

**信任的交接**：沈砚之离开前把他能做的全部做完，剩下的全部留给苏锦婳。这是两人同盟的又一次确认——他走，不是甩手，是把能交接的全部交接。

### 四字节奏

[契书落笔] → [账目分明] → [织娘立规] → [石青初现] → [砚之远行]

## 版本B：暗流涌动

**基调**：沈砚之即将离开，沈老栓在这最后窗口期突然现身——他不是来送行的，是来确认儿子走了之后，这间铺子到底谁说了算。

### 核心情节点

**情节点1** [沈老栓的最后一访]

**沈老栓线**
- **沈老栓状态**：锦云丝局门口，没有进门，站在街对面看着
- **核心事件**：沈砚之收拾行囊时，沈老栓来了。他没有进后院，只在前厅站了一会儿，看了一眼柜上的契纸和账册，然后问："三成的事，我还没签字。"
- **冲突**：三成干股是沈砚之和苏锦婳谈好的，但沈老栓要的是"落袋"——他要看到字据，或者要一个说法
- **反转**：沈砚之说："爹，等我回来再签。我走了，这事不急在今天。"
- **钩子**：沈老栓看了他一眼，没有再提这件事

**情节点2** [定制收·第一张契书]

**苏锦婳线**
- **女主状态**：锦云丝局，周老汉来签第一张定制收契书
- **核心事件**：周老汉带了五户蚕农的茧子来，总共五十多斤。苏锦婳当场开箱验质——按沈砚之定的三等标准，分出甲乙丙
- **冲突**：周老汉嫌乙等被压了一等，说"上次吴掌柜给我开的也是乙等"
- **反转**：苏锦婳把丝抽出来让他自己看——光泽、匀度、断头数，三个指标当面验完，周老汉服了
- **钩子**：第一张定制收契书签订。五十斤鲜茧，苏锦婳付了定金三百文

**情节点3** [街角骚扰·麻六登场]

**苏锦婳线**
- **女主状态**：锦云丝局门口，有人来收"场地费"
- **核心事件**：一个自称"麻六"的男人带着两个跟班来，问丝局收不收"看护费"——十两银子一年，不交就别想在镇上做生意
- **冲突**：苏锦婳第一次遇到地头蛇，她知道硬碰不行
- **反转**：陈四拦住麻六，说"这是沈相公家的铺子，沈砚之是府学秀才"，麻六哼了一声，说"穷酸秀才我见多了"
- **钩子**：麻六走时看了一眼柜台上的丝，说："行啊，生意不错。"——威胁的意思很清楚

**情节点4** [沈砚之的反制·临走前的一手]

**沈砚之线**
- **男主状态**：沈砚之从外头回来，直接去找了镇上的李亭首
- **核心事件**：他没有告诉苏锦婳，直接拿了自己的秀才帖子，找到镇上的亭首——说铺子新开，有人来收"看护费"，请亭首留意
- **冲突**：亭首说"麻六是镇东头的，你们在镇西，两不相干"
- **反转**：沈砚之笑了一下，说："两不相干，他来收什么钱？"
- **钩子**：亭首沉默了一会儿，说"我知道了"

**情节点5** [砚之背影·独自开局]

**苏锦婳线**
- **女主状态**：沈砚之走后第一天清晨，苏锦婳独自坐在柜台前
- **核心事件**：账册在手上，契书在抽屉里，定制收第一单已经签了，麻六的威胁还在耳边。她把账册翻到第一页，开始算：五十斤鲜茧，按定制收条款，这一单能赚多少
- **钩子**：门外那个乞丐又出现了，还是站在墙角，没有进来

### 核心台词（3-5句）

1. 沈老栓："三成的事，我还没签字。"
2. 沈砚之："爹，等我回来再签。"
3. 麻六："穷酸秀才我见多了。"
4. 沈砚之（对亭首）："两不相干，他来收什么钱？"
5. 苏锦婳（内心）："他走之前，把能做的都做了。剩下的，该我来了。"

### 情绪曲线

[压抑→紧绷→冲突→暗中解决→淡淡的焦虑]

沈老栓的来访是压力，麻六的威胁是更直接的压力，沈砚之的暗中反制是化解，但苏锦婳知道：她要独自面对了。

### 主题落点

**保护与独立的张力**：沈砚之想保护她，但他知道不能永远保护。这一章是苏锦婳从"被保护"到"独自应对"的转折点。

### 四字节奏

[老栓来访] → [定制签单] → [麻六骚扰] → [砚之反制] → [独自开局]

## 版本C：石青·等待的人

**基调**：石青出场为主视角，穿插沈砚之离开前的交代。以"等一个机会"为核心意象，串联石青与苏锦婳两个人的等待。

### 核心情节点

**情节点1** [石青：镇西街角的乞丐]

**石青线**
- **石青状态**：镇西街角，锦云丝局对面的墙根
- **核心事件**：石青已经在这个位置站了三天。他不讨饭，不惹事，就只是看着这间铺子——每天人来人往，蚕农进出，丝局开张
- **冲突**：他注意到一个细节——这间铺子没有"看护"，门口连个看门的都没有。而镇上其他铺子，多少都要给麻六交钱
- **反转**：他看到麻六来收钱被挡了回去。他看到那个女人（苏锦婳）站在柜台后头，不卑不亢地应对
- **钩子**：他注意到这间铺子的女人，和别人不一样

**情节点2** [苏锦婳：施舍的饭]

**苏锦婳线**
- **女主状态**：锦云丝局门口，傍晚
- **核心事件**：苏锦婳发现那个乞丐还在。她让沈诚之把剩下的饭端出去，放在他面前
- **冲突**：对方没有接，问了一句："为什么给我？"
- **反转**：苏锦婳说了一句："因为你没有伸手。"
- **钩子**：石青的眼神变了。他接过了饭，但没有走，还站在原地

**情节点3** [砚之的最后交代]

**沈砚之线**
- **男主状态**：锦云丝局内，行囊已经收好
- **核心事件**：沈砚之把定制收契书、账册、一张写好地址的纸条放在抽屉里，对苏锦婳说："江宁府学在祟文街，我住在学舍东边的茂源客栈。有事写信。"
- **冲突**：他犹豫了一下，没有说出口的话是——"如果你遇到危险怎么办"
- **反转**：沈砚之补了一句："那个乞丐——如果他愿意，让他看门。"
- **钩子**：苏锦婳看了他一眼，没有问为什么

**情节点4** [麻六的威胁]

**苏锦婳线**
- **女主状态**：第二天上午，麻六再来
- **核心事件**：麻六再来时，带了多一个人。他说"十两银子是上头的价，今天再加五两手续费"
- **冲突**：苏锦婳知道这是恐吓，但她刚开业，账上没有余钱
- **反转**：门口那个乞丐忽然站直了身子，往铺子门口走了一步——没有说话，就只是站着
- **钩子**：麻六看了他一眼，带着人走了

**情节点5** [砚之远行·石青入局]

**石青线 + 苏锦婳线**
- **核心事件**：沈砚之启程。石青站在街对面，看着他的背影消失。然后他走进铺子，问："你们这儿，还缺人手吗？"
- **核心台词**：苏锦婳问："你会什么？"石青答："我会等。"苏锦婳看了他一眼，问："等什么？"他说："等一个机会。"
- **钩子**：苏锦婳说："门口有个乞丐，每天都在。你去问他，要不要来铺子里吃饭。"

### 核心台词（3-5句）

1. 苏锦婳："因为你没有伸手。"
2. 沈砚之："那个乞丐——如果他愿意，让他看门。"
3. 麻六："十两银子是上头的价，今天再加五两手续费。"
4. 石青："我会等。"
5. 苏锦婳："门口有个乞丐，每天都在。你去问他，要不要来铺子里吃饭。"

### 情绪曲线

[观察→接触→交代→冲突→入局]

石青的等待、苏锦婳的等待、沈砚之临走前的安排——三个人都在等，但等的东西不一样。石青等机会，苏锦婳等沈砚之走后的第一场硬仗，沈砚之等中举的消息。

### 主题落点

**等待的意义**：每个人都在等。石青在等一个不死的理由，苏锦婳在等一个站稳脚跟的机会，沈砚之在等一个改变命运的结果。这一章的核心意象是"等待"，而"机会"是打开等待的钥匙。

### 四字节奏

[石青观察] → [苏锦婳施饭] → [砚之交代] → [麻六威胁] → [石青入局]

本章容量约为1章，主要内容：
- 沈砚之离开前的各项交代（契书/账目/关系）
- 定制收模式正式启动
- 石青/麻六的出场铺垫

- **第27章**：砚之远行 + 定制收第一单 + 麻六骚扰/石青出现
